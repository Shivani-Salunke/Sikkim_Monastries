## Installation & Setup

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd sikkim-monastery-platform
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`
